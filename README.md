# 한글그림-한글공방

한글을 입력하면 아름다운 그림으로 변환해주는 웹 애플리케이션입니다.

## 기능

- 한글 입력 시 실시간 그림 생성
- 다양한 그림꼴 스타일 선택 (색동, 조각보, 물수제비, 한울림, 입방체, 단청, 만화경, 세모)
- 생성된 이미지 다운로드 기능
- PC 및 모바일 지원

## 배포 방법

### Netlify로 배포하기

1. [Netlify](https://www.netlify.com/)에 가입하기
2. "Add new site" > "Import an existing project" 클릭
3. GitHub 저장소 연결
4. 배포 설정은 자동으로 `netlify.toml` 파일을 읽습니다
5. "Deploy site" 클릭

### GitHub Pages로 배포하기

1. GitHub에 저장소 생성
2. Settings > Pages 메뉴로 이동
3. Source를 "main" 브랜치로 설정
4. Save 클릭

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- jQuery
- html2canvas (이미지 캡처)
